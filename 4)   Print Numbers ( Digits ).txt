🧩 Exercise: print_numbers

📜 Description:
Write a function that displays all digits from 0 to 9 using only the write() function.

🧠 Instructions:
- Function name: print_numbers
- Takes no parameters and returns nothing.
- Only the write() function is allowed.
- Include <unistd.h>.

🧩 Prototype:
void    print_numbers(void);

⚙️ Example Usage:
#include <unistd.h>

void    print_numbers(void);

int main(void)
{
    print_numbers();
    return (0);
}

Expected Output:
0123456789

💡 Solution:
#include <unistd.h>

void    ft_print_numbers(void)
{
    write(1, "0123456789", 10);
}
