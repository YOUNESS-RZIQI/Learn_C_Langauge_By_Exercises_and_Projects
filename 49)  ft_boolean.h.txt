Description:  
Create a header file that defines a custom boolean type and related macros for clarity and convenience.  
This header provides macros for `TRUE`, `FALSE`, and checks if a number is even, along with message constants.

- `#include <unistd.h>`

- `typedef int t_bool;`

- `#define TRUE 1` / `#define FALSE 0`  
  â†’ Defines constants to represent true and false values for readability.  

- `#define EVEN(nbr) (nbr % 2 == 0)`  
  â†’ A macro that returns true if `nbr` is even, otherwise false.  

- `#define EVEN_MSG` / `#define ODD_MSG`  
  â†’ Message macros that can be printed depending on whether the argument count is even or odd.
